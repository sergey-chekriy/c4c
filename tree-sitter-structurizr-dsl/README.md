# tree-sitter-structurizr-dsl

Tree-sitter grammar for the Structurizr DSL used by `c4c`.

`grammar.js` is the syntax source of truth. `src/parser.c`, `src/grammar.json`,
`src/node-types.json`, and `src/tree_sitter/` are generated by Tree-sitter and
committed so normal Rust builds do not require Node.js.

Node.js is needed only when changing the grammar. Install the development
dependency and regenerate from this directory:

```bash
npm install
npm run generate
npm test
```

The equivalent repository-level commands are `make grammar` and
`make grammar-test`. Commit generated artifact changes with the grammar change.
